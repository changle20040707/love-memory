# 照片目录说明

请将你们的旅游照片放在这个目录中。

## 照片命名建议

- 使用有意义的名称，如：`hangzhou-west-lake.jpg`
- 建议使用小写字母和连字符
- 避免使用中文或特殊字符

## 照片格式

- 推荐格式：JPG / JPEG
- 也支持：PNG, WebP, GIF

## 照片大小建议

- **分辨率**：建议宽度 1200-2000px
- **文件大小**：每张建议 200KB-500KB，不要超过 1MB
- **压缩工具**：可以使用 [TinyPNG](https://tinypng.com/) 压缩图片

## 如何添加照片

1. 将照片复制到此目录
2. 打开 `data/config.json`
3. 在 `photos` 数组中添加照片信息：

```json
{
  "id": 1,
  "src": "images/photos/你的照片文件名.jpg",
  "title": "照片标题",
  "location": "拍摄地点",
  "date": "2024-05-01",
  "description": "照片描述",
  "category": "旅行"
}
```

## 分类说明

可以自定义分类，常见的有：
- 旅行
- 纪念日
- 日常
- 约会
- 生日
